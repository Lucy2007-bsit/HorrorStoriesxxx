# XXXGhostly💀

## 💀 Description

This project, titled "XXXGhostly," is a surprisingly unsettling web experience built primarily with JavaScript, HTML, and CSS. Despite its somewhat humorous repository name and description, the actual content of the website is designed to be a horror-themed interactive story. It aims to create a creepy and immersive atmosphere through visual effects, sound, and text manipulation, leveraging a dark theme and great stories. The website utilizes a single HTML entry point (`index.html`) and relies heavily on client-side JavaScript for its dynamic and terrifying elements.

## ✨ Features

*   **Immersive Horror Experience:** Utilizes a dark theme, unsettling text, and ambient sound to create a creepy atmosphere. 👻
*   **Reactive Storytelling:** Utilizes interactive event triggers to mutate the text, creating an unsettling, evolving creepypasta that adapts to how the user navigates the page. 😱
*   **Interactive Storytelling:** Presents a narrative in Filipino that becomes increasingly surreal and disturbing as the user scrolls. 📖
*   **Visual Effects:** Employs techniques like screen flickering, red flashes, and shadow effects to enhance the horror. ⚡
*   **Audio Integration:** Incorporates scream sound effects to punctuate scares. 🔊
*   **Responsive Design:** Adjusts behavior and triggers based on screen size (mobile vs. desktop). 📱
